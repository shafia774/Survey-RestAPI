## SetUp

```
- composer install
- cp .env.example .env
- php artisan key:generate
- npm install
- php artisan migrate

```

- while in development

    ```
    npm run dev 
    php artisan serve

    ```

- for production

        npm run dev 

- the end it 

        npm run build   

- To set up admin

        php artisan create-admin
